# LawyerWebsite
Website for a Lawyer. Tech-Stack will be FastAPI, Jinja2, Bulma CSS and HTMX. 

# Usage
1. Clone Repo
2. Create venv
   `uv venv -p 3.13`
3. Run App
   `uv run` or activate venv and run `python main.py`
